## 구성품

- **주사위 40개**: 플레이어 색상별 8개씩 (5색)
- **카지노 카드 6장**: 숫자 1~6이 적힌 카지노
- **지폐 54장**: $10,000 ~ $90,000
- **선 플레이어 마커**: 1개

## 게임 준비

1. 6개의 카지노 카드를 테이블 중앙에 1~6 순서로 배치
2. 각 플레이어는 자신의 색상 주사위 8개를 받음
3. 지폐를 섞어서 덱으로 준비
4. 선 플레이어 결정

## 라운드 시작 (배당금 배치)

각 라운드 시작 시:
1. 지폐 덱에서 카드를 뽑아 각 카지노 앞에 배치
2. 각 카지노에 **최소 $50,000 이상**이 될 때까지 지폐를 추가
3. 한 카지노에 여러 장의 지폐가 놓일 수 있음

## 게임 진행

### 플레이어 턴

1. **주사위 굴리기**: 자신이 가진 모든 주사위를 한 번에 굴림
2. **숫자 선택**: 나온 숫자 중 하나를 선택
3. **주사위 배치**: 선택한 숫자가 나온 주사위를 **모두** 해당 숫자의 카지노에 배치
   - 예: 주사위 8개를 굴려서 3이 3개 나왔다면, 3번 카지노에 3개 모두 배치해야 함
4. 남은 주사위가 있으면 다음 턴에 다시 굴림
5. 모든 플레이어가 주사위를 다 배치할 때까지 반복

### 핵심 규칙: 동점 처리 (가장 중요!)

**같은 카지노에 동일한 개수의 주사위를 놓은 플레이어들은 서로 상쇄되어 무효 처리됨**

예시:
- A 플레이어: 3개, B 플레이어: 3개, C 플레이어: 2개
- → A와 B는 상쇄되어 무효, C가 1등으로 배당금 획득

## 라운드 정산

1. 각 카지노별로 가장 많은 주사위를 놓은 플레이어가 **가장 높은 금액의 지폐** 획득
2. 2등은 두 번째로 높은 금액 획득 (지폐가 2장 이상일 경우)
3. 동점자는 모두 상쇄 → 다음 순위 플레이어에게 기회
4. 정산 후 모든 주사위를 플레이어에게 반환

## 게임 종료

- **총 4라운드** 진행
- 4라운드 종료 후 가장 많은 돈을 모은 플레이어 승리

## 2인 플레이 특별 규칙

- 각 플레이어는 자신의 색상 주사위 8개 + **중립 색상 주사위 4개** 사용
- 중립 주사위도 함께 굴리고 배치해야 함
- 중립 주사위는 상쇄 효과에 영향을 줌 (블로킹 용도)

## 전략 팁

1. **상쇄 활용**: 상대방과 같은 개수로 맞춰서 상쇄시키기
2. **분산 투자**: 한 카지노에 올인하지 말고 여러 곳에 분산
3. **후반 관망**: 먼저 배치하면 상대에게 정보를 줌
4. **고액 노리기**: $90,000, $80,000 등 고액 카지노 집중 공략
5. **소수 주사위 활용**: 적은 수의 주사위로 빈 카지노 차지
